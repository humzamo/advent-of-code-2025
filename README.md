## Running Solutions
To run solutions for a specific day, run the command:
```
make run day=`X`
```
where X is an integer between 1 and 12.

## Running Tests
To run all tests in the repo, run the command:
```
make test-all
```


To run tests for a specific day, run the command:
```
make test day=`X`
```
where X is an integer between 1 and 12.
